# FriendMap — Инструкция по настройке

## Что нужно сделать (5 минут)

### 1. Создать Firebase проект (бесплатно)

1. Зайди на https://console.firebase.google.com
2. Нажми **"Создать проект"** → введи любое имя → создай
3. В левом меню выбери **Realtime Database** → **Создать базу данных**
   - Выбери регион (europe-west1 для России)
   - Режим: **Тестовый** (на 30 дней, потом можно продлить)

### 2. Получить конфиг

1. В левом меню нажми ⚙️ **Настройки проекта**
2. Прокрути вниз до **"Ваши приложения"** → нажми **</>** (Web)
3. Введи имя приложения → **Зарегистрировать**
4. Скопируй объект `firebaseConfig`

### 3. Вставить конфиг в index.html

Найди в `index.html` строку:
```js
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  ...
```

Замени весь объект на скопированный из Firebase.

### 4. Правила базы данных (опционально, для безопасности)

В Firebase Console → Realtime Database → **Правила**:
```json
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
```
> Для тестирования оставь тестовый режим (read/write: true)

### 5. Открыть приложение

Просто открой `index.html` в браузере — всё готово!

---

## Функции приложения

- 🗺️ **Карта мира** — интерактивная карта OpenStreetMap
- 📍 **Поделиться локацией** — отправить своё местоположение
- 👥 **Друзья** — добавить друга по имени пользователя
- 💬 **Чат** — личные сообщения с каждым другом в реальном времени
- 🔄 **Авто-обновление** — локации и сообщения обновляются мгновенно
- 💾 **Авто-вход** — запоминает сессию в браузере
